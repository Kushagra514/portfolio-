## 🔧 Customization

### Change Accent Color
In `cosmic.css`, replace `#00d9ff` with your color:
```css
--neon-blue: #your-color-here;
```

### Adjust Animation Speed
In `cosmic.css`:
```css
--cosmic-transition: 0.4s; /* was 0.4s */
--cosmic-slow: 0.8s; /* was 0.8s */
```

### Modify Glow Intensity
In `cosmic.css`, change box-shadow opacity:
```css
box-shadow: 0 0 30px rgba(0, 217, 255, 0.5); /* increase second number for brighter */
```

### Disable Specific Animations
In `cosmic-universe.js`, comment out:
```javascript
// initParticleSystem(); // Disable particles
// initCosmicCanvas(); // Disable starfield
// initOrbitals(); // Disable orbits
```
