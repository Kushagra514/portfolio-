## 🔧 Customization

**Make faster**: Edit `0.7s` → `0.5s` in CSS
**Make slower**: Edit `0.7s` → `1s` in CSS
**Change color**: Replace `rgba(200, 169, 110, ...)` 
**Disable parallax**: Comment out `initParallax()` in JS

See `ANIMATION_GUIDE.md` for details.
