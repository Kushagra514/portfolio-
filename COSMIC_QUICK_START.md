## ⚡ Setup (30 seconds)

### Option A: Use Complete Template (Easiest)
```bash
cd frontend
cp cosmic-portfolio.html index.html
# Done! Cosmic theme active.
```

### Option B: Add to Existing HTML
In `<head>`:
```html
<link rel="stylesheet" href="cosmic.css" />
```

Before `</body>`:
```html
<script src="cosmic-universe.js"></script>
```

## ⚙️ Customization

### Change Glow Color
Find in `cosmic.css`:
```css
--neon-blue: #00d9ff
```
Replace with your color.

### Slower/Faster Animations
In `cosmic.css`, change timing:
```css
animation: floatNebula 20s ease-in-out infinite;
/* Change 20s to 30s (slower) or 15s (faster) */
```

### Reduce Particle Count
In `cosmic-universe.js`:
```javascript
const particleCount = 30; // Change to 15 or 50
```
