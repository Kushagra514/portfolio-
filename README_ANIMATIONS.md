## 🎯 Customization

### Make Animations Faster
In `animations.css`:
```css
/* Change 0.7s → 0.5s or 0.3s → 0.2s */
```

### Make Animations Slower
In `animations.css`:
```css
/* Change 0.7s → 1s or 0.3s → 0.5s */
```

### Change Gold Color
Find: `rgba(200, 169, 110, ...)`
Replace with your color in RGBA format

### Disable Parallax
In `animations.js`, comment out:
```javascript
// initParallax();
```
